#### ✅ JSON이란?

- **텍스트 기반의 데이터 형식**
- **key-value 쌍**으로 구성됨  
  → 예: `{"name": "John", "age": 30}`
- 시스템 간 데이터 교환에 사용
- 사람이 읽기 쉽고, 다양한 언어에서 지원됨

---

#### ✅ Gson이란?

- Java에서 **JSON ↔ 객체** 변환을 쉽게 처리하는 라이브러리
- **직렬화**: Java 객체 → JSON
- **역직렬화**: JSON → Java 객체
- Google에서 제공, 설정 없이 사용 가능

---

#### ✅ 관계 정리

| 개념  | 설명 |
|-------|------|
| JSON  | 데이터를 주고받는 **형식** |
| Gson  | Java에서 JSON을 **다루기 위한 도구** |
