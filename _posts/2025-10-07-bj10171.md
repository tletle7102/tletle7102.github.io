### 백준 코딩테스트 연습

#### 백준 10171번 입출력과 사칙연산

##### 문제
아래 예제와 같이 고양이를 출력하시오.

##### 입력
없음.

##### 출력
고양이를 출력한다.

##### 예제 입력
```plaintext

```

##### 예제 출력
```plaintext
\    /\
 )  ( ')
(  /  )
 \(__)|
```

##### 제출 답안
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```
